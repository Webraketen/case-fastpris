# Case study — Fastpris Städ Göteborg

How [Fastpris Städ Göteborg](https://fastprisstad.se) ships kontorsstäd, företagsstäd and flyttstäd in Göteborg to fast pris. Built and maintained by [Webraketen](https://webraketen.se).

## Stack

- **Astro** for static landing pages
- **Cloudflare** in front for analytics + bot fight
- **Netlify** for hosting + form functions
- **Resend** for transactional confirmations

## What Fastpris Städ does

Kontorsstäd, företagsstäd, lokalvård, hemstäd and flyttstäd in Göteborg to fast pris from start. Kollektivavtal, F-skatt, RUT-avdrag. Offert inom 24 timmar.
